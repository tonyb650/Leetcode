### 504. Base 7

**Problem**\
Given an integer num, return a string of its base 7 representation.

**Example 1:**\
Input: num = 100\
Output: "202"

**Example 2:**\
Input: num = -7\
Output: "-10"

**Constraints:**
- -107 <= num <= 107

### Solution, javascript

```javascript
var convertToBase7 = function(num) {
    if (num === 0) return "0";
    let result = "";
    let val = 0;
    let base = 7;
    let isNeg = num < 0 ? true : false;
    num = Math.abs(num);
    while(num != 0){
        val = num % base
        num = (num - val) / base
        result = val + result
    }
    if(isNeg) result = "-" + result;
    return result;
};
```
